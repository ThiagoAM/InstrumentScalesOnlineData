---
name: Pivot-Point Heist in A Dorian
description: Steal more melodic solos by sneaking away from a target note and returning with rhythm, tension, and control.
---
### Mission
Tonight you are a phrase thief with manners. Work in **A Dorian** (A B C D E F# G) at **80-92 bpm** and pick one fretboard area you know well.

Your target note is **C**. Every phrase must visit it, leave it, and come back without sounding like a scale worksheet.
<!-- block -->
### Lock the vault first
Play four tiny phrases of **3 to 5 notes** that all **start on C**. Keep the rhythm simple: only **eighth notes and quarter notes**.

After each phrase ask:
- Did C sound intentional or accidental?
- Did the line still sound like A Dorian?
- Could I sing it back immediately?
<!-- block -->
### The getaway rule
Now each phrase must **leave C by step**, then **jump back by a skip**. Example idea: move from C to B or D, then answer with A, E, or G.

Make **five different 2-bar phrases**. No phrase may end on the same note twice in a row. That forces you to hear multiple exits from the same pivot point.
<!-- block -->
### Add the disguise
Repeat the game, but now one note in each phrase must get either:
- an accent,
- a short rest before it, or
- a longer hold than the surrounding notes.

This is the disguise. The scale notes stay familiar, but the phrasing should feel sneakier and more musical.
<!-- block -->
### Boss round: the heist loop
Improvise over **Am7 to D7** for **six rounds of 2 bars**. In every round:
1. touch **C** once,
2. leave it in a new way,
3. return with a different rhythm than before.

If two rounds sound like twins, throw one out and invent a riskier escape route.
<!-- block -->
### Debrief
Ask yourself:
- Which return to C sounded coolest?
- Did my accents create shape or just volume?
- When did I stop sounding like a scale exercise?

Final upgrade: run the same heist in **D Dorian** without changing the rules. If the phrasing still feels alive, you got away clean. 🎸
